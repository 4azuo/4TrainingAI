# Danh sách màn hình (Screen List)

> Tài liệu **dùng chung**: liệt kê mọi màn hình của RestaurantApp. Mỗi màn hình có 1 file
> **basic design** (`03-basic-designs/`) và 1 file **detail design** (`04-detail-designs/`).

| Mã | Tên màn hình | Form (Views) | Chức năng chính | Basic | Detail |
|----|--------------|--------------|-----------------|-------|--------|
| **SCR-01** | Quản lý & Tìm kiếm món | `MenuForm` | Thêm / sửa / xoá / tìm món | [basic](../03-basic-designs/SCR-01-quan-ly-mon.md) | [detail](../04-detail-designs/SCR-01-quan-ly-mon.md) |
| **SCR-02** | Gọi món / Lập đơn | `MainForm` | Chọn món → thêm vào đơn → tính tổng tiền | [basic](../03-basic-designs/SCR-02-goi-mon.md) | [detail](../04-detail-designs/SCR-02-goi-mon.md) |

## Quy ước mã màn hình
- Mã dạng `SCR-NN` (NN tăng dần). Mã **bất biến** — đổi tên màn hình vẫn giữ mã.
- Tên file design trùng mã + slug tiếng Việt không dấu: `SCR-01-quan-ly-mon.md`.
