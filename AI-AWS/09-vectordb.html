🧮 Embeddings & Vector Stores

Embedding = biến chữ thành vector số. Văn bản gần nghĩa → vector gần nhau. Vector store lưu & tìm "hàng xóm gần nhất" — trái tim của RAG & tìm kiếm ngữ nghĩa.

Model tạo embedding trên AWS

ModelGhi chú
Amazon Titan Text EmbeddingsEmbedding văn bản đa ngôn ngữ, gọi qua Bedrock.
Amazon Titan Multimodal EmbeddingsNhúng ảnh + chữ chung một không gian → tìm ảnh bằng chữ.
Cohere EmbedCó trên Bedrock, đa ngôn ngữ, chất lượng cao.

Lưu vector ở đâu trên AWS?

OpenSearch Serverless

Mặc định phổ biến

Vector engine (k-NN), managed, tích hợp sẵn với Bedrock Knowledge Bases.

Aurora PostgreSQL (pgvector)

Đã có Postgres

Thêm cột vector vào CSDL quan hệ sẵn có — không phải dựng hệ thống mới.

Amazon Neptune Analytics

Đồ thị + vector

Kết hợp graph & tìm kiếm vector cho dữ liệu liên kết.

MemoryDB / Redis · Pinecone

Độ trễ thấp / bên thứ ba

Tìm vector cực nhanh trong bộ nhớ; Pinecone là lựa chọn managed ngoài AWS được Bedrock hỗ trợ.
