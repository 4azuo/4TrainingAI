🟧 Amazon Bedrock — Nền tảng Generative AI

Bedrock = "cửa hàng FM" serverless. Một API duy nhất gọi nhiều Foundation Model của nhiều nhà cung cấp; không quản máy chủ, không huấn luyện từ đầu.

🧩 Ví dụ tích hợp — Chatbot CSKH serverless
API Gateway → Lambda → Bedrock (Claude, Converse API) → trả lời; lịch sử hội thoại lưu DynamoDB; prompt template & key trong Secrets Manager; log token ra CloudWatch.
📌 Converse API là API hội thoại thống nhất (multi-turn, tool use, ảnh) — nên dùng thay cho InvokeModel thô vì cùng một cấu trúc cho mọi model.
