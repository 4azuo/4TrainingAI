☁️ Tổng quan hệ sinh thái AI của AWS

AWS chia dịch vụ AI thành 3 tầng: dịch vụ AI dùng-ngay (API), nền tảng GenAI Bedrock, và nền tảng ML SageMaker cho người tự xây mô hình.

3 tầng AI/ML của AWS

① AI Services — gọi API, không cần biết ML

Mô hình đã huấn luyện sẵn, trả lời qua API. Ví dụ: RekognitionTextractComprehend TranscribePollyTranslateLexKendra

② Generative AI — Amazon Bedrock & Amazon Q

Truy cập Foundation Models (FM) qua 1 API thống nhất; thêm RAG (Knowledge Bases), Agents, Guardrails. Amazon Q là trợ lý GenAI dựng sẵn. BedrockAmazon Q

③ ML Platform — Amazon SageMaker

Tự build / train / fine-tune / deploy mô hình; SageMaker JumpStart cung cấp FM mã nguồn mở để tùy biến sâu. SageMakerJumpStart
