🌀 Vibe Coding là gì?

"Vibe coding" — bạn mô tả ý tưởng bằng ngôn ngữ tự nhiên, AI viết code, bạn xem kết quả chạy và tiếp tục mô tả để chỉnh — thay vì tự gõ từng dòng.

Nguồn gốc thuật ngữ

Khái niệm do Andrej Karpathy (đồng sáng lập OpenAI) đặt tên đầu 2025: "fully give in to the vibes… and forget that the code even exists" — phó mặc cho cảm hứng, tin vào AI, gần như quên rằng có code bên dưới. Từ đó "vibe coding" trở thành cách gọi chung cho lối lập trình dẫn dắt bằng prompt.

💡 Điểm cốt lõi: con người chuyển vai từ "người gõ code" sang "người mô tả ý định + kiểm duyệt kết quả". AI lo phần cú pháp, bạn lo phần ý tưởng và đánh giá.

3 cấp độ "vibe" — chọn đúng mức cho việc của bạn

① Tự động hoàn thiện (Autocomplete)

AI gợi ý dòng tiếp theo khi bạn gõ. Bạn vẫn kiểm soát hoàn toàn. VD: GitHub Copilot, Cursor Tab.

② Trợ lý theo lệnh (Inline / Chat)

Bạn ra lệnh "viết hàm X", "sửa lỗi này" cho một vùng code. VD: Cmd+K của Cursor, Copilot Chat.

③ Agent tự chủ (Vibe thuần)

AI tự lập kế hoạch, sửa nhiều file, chạy lệnh, tự sửa lỗi. VD: Cursor Agent, Claude Code, Lovable, Replit Agent.
