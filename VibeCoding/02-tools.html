🧰 Bản đồ công cụ Vibe Coding

Các tool chia thành 4 nhóm theo nơi bạn làm việc: IDE thông minh, Text-to-App (prompt ra cả app), Agent dòng lệnh (CLI), và Cloud/Autonomous. Biết nhóm nào hợp việc gì quan trọng hơn nhớ tên từng tool.

① IDE thông minh — viết code có AI hỗ trợ

Bạn vẫn ở trong một trình soạn thảo code đầy đủ, AI là lớp trợ lý phủ lên trên.

IDE

Cursor

Bản fork của VSCode, AI-first. Có Tab (autocomplete mạnh), Cmd+K (sửa inline), Chat hiểu cả codebase, và Agent (Composer) sửa nhiều file. Trả phí theo tháng.

IDE

Windsurf

IDE AI của Codeium, nổi bật với agent "Cascade" có khả năng theo dõi ngữ cảnh và tự chạy nhiều bước.

Extension

GitHub Copilot

Cắm vào VSCode / JetBrains / Visual Studio. Autocomplete + Chat + chế độ Agent. Quen thuộc, tích hợp sâu GitHub.

Open-source

Cline / Roo Code / Continue

Extension mã nguồn mở cho VSCode. Bạn tự cắm API key của model (Claude, GPT…). Kiểm soát chi phí & dữ liệu tốt hơn.

② Text-to-App — prompt ra cả ứng dụng (3rd-party platform)

Bạn gõ mô tả trong trình duyệt, nền tảng dựng cả app full-stack kèm preview, thường có deploy & Git tích hợp.

Text-to-App

Lovable

Mô tả → app web full-stack (React + Supabase). Preview trực tiếp, đồng bộ GitHub, deploy 1 click. Hợp người không chuyên dựng MVP nhanh.

Text-to-App

Bolt.new (StackBlitz)

Chạy full-stack ngay trong trình duyệt (WebContainers). Sửa code được, deploy lên Netlify. Nhanh để thử ý tưởng.

UI gen

v0 (Vercel)

Thiên về sinh giao diện: React + Tailwind + shadcn/ui từ prompt hoặc ảnh. Deploy thẳng lên Vercel.

Cloud IDE

Replit Agent

IDE trên cloud + agent dựng và deploy app, có sẵn hosting/DB. Làm việc hoàn toàn trên trình duyệt, kể cả trên điện thoại.

③ Agent dòng lệnh (CLI) — AI làm việc trong terminal/repo của bạn

CLI Agent

Claude Code

Agent của Anthropic chạy trong terminal/IDE, thao tác trực tiếp trên repo: đọc/sửa file, chạy test, commit, tạo PR. Mạnh cho codebase thật.

CLI Agent

Gemini CLI / Codex CLI

Các agent dòng lệnh tương tự từ Google / OpenAI. Cùng triết lý: AI sống trong môi trường dev của bạn.

Autonomous

Devin (Cognition)

"Kỹ sư AI" tự chủ: nhận task, tự lập kế hoạch, code, test trong môi trường cloud riêng. Hướng tới giao việc trọn gói.

✅ Quy tắc chọn nhanh: cần app demo gấp & không rành code → Lovable/Bolt/v0. Đang có codebase & muốn kiểm soát → Cursor/Copilot/Claude Code. Tự động hóa task lập trình lặp lại → CLI agent.
