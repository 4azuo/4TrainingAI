💻 Dùng AI trên VSCode / IDE — Ưu & Nhược

Đây là cách "vibe coding" gắn với môi trường dev thật: AI là extension/agent phủ lên IDE, còn code, Git, terminal, test… vẫn là của bạn và nằm trên máy bạn.

Các lựa chọn trên VSCode

Claude CodeCLI + tích hợp IDEAgent thao tác repo trực tiếp; chạy được trong terminal của VSCode.

✅ Ưu điểm

  • Toàn quyền với code — file nằm trên máy bạn, sửa/đọc/refactor tùy ý.
  • Git, test, debug, terminal đầy đủ — đúng quy trình dev chuyên nghiệp.
  • Tích hợp hệ sinh thái — extension, linter, formatter, debugger sẵn có.
  • Linh hoạt model — đổi/tự cắm model, kể cả model chạy local cho dữ liệu nhạy cảm.
  • Hiểu ngữ cảnh dự án lớn tốt hơn nền tảng text-to-app.
  • Mở rộng ngoài web — backend, script, data, mobile, DevOps… không bó vào web app.

❌ Nhược điểm

  • Cần biết lập trình ở mức cơ bản để dùng hiệu quả & review.
  • Phải tự lo môi trường — cài đặt, dependency, cấu hình.
  • Không "1-click deploy" — tự dựng pipeline triển khai.
  • Dễ "tin mù" gợi ý AI — chấp nhận code sai/không an toàn nếu không review.
  • Quản lý chi phí token/quyền truy cập cần chú ý, nhất là theo nhóm.
