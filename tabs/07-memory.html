🧠 Memory — Bộ nhớ của AI

Memory là khả năng ghi nhớ — giúp AI nhớ thông tin qua nhiều phiên làm việc, không bị mất trí nhớ sau mỗi cuộc trò chuyện.

4 Loại Memory

⚡ SHORT-TERM MEMORY (Working Memory)

Nhớ trong phiên hiện tại — chính là Context Window
"Bạn vừa nói muốn viết email cho khách VIP..."

💾 LONG-TERM MEMORY

Lưu vào database, nhớ qua nhiều phiên
"Bạn thích tone văn bản trang trọng..."

📖 EPISODIC MEMORY

Nhớ sự kiện, tương tác đã xảy ra
"Tuần trước bạn đã hỏi về chiến dịch X..."

📚 SEMANTIC MEMORY

Nhớ kiến thức, thông tin được nạp vào
"Sản phẩm A có giá 500k, tính năng là..."
