⚙️ Workflow / Pipeline — Quy trình tự động

Workflow là dây chuyền tự động — kết nối Hook + Agent + Tool + Skill thành quy trình chạy liên tục không cần con người can thiệp từng bước.

4 Workflow Patterns

1️⃣ LINEAR (Tuần tự)

A → B → C → D

Mỗi bước cần kết quả của bước trước

2️⃣ PARALLEL (Song song)

A → [B, C, D] → E

Các task độc lập, chạy đồng thời để nhanh hơn

3️⃣ CONDITIONAL (Điều kiện)

A → Điều kiện? → B hoặc C

Xử lý khác nhau tùy tình huống

4️⃣ LOOP (Vòng lặp)

A → B → Đạt chưa? → C hoặc lại A

Tinh chỉnh đến khi đạt chất lượng

Ví dụ: cùng một việc, 3 cách dựng

Bài toán: "Có email khách hàng mới → AI tóm tắt & phân loại → báo vào Slack". Cùng workflow này dựng được ở cả 3 cấp độ:

① Zapier — No-code (kéo thả, không viết code)

Một "Zap" = 1 Trigger + các Action nối tiếp. Bạn chọn từ menu, không lập trình:

# Pattern: LINEAR (tuần tự)
TRIGGER : Gmail — "New Email" (lọc theo nhãn "khách hàng")
   ↓
ACTION 1: ChatGPT (Zapier tích hợp sẵn)
          Prompt: "Tóm tắt email sau trong 2 câu và phân loại
                   [Khiếu nại / Hỏi giá / Hỗ trợ]: {{email_body}}"
   ↓
ACTION 2: Slack — "Send Channel Message" tới #cskh
          Nội dung: {{tóm tắt}} · Loại: {{phân loại}}

👉 Toàn bộ làm bằng giao diện kéo thả — hợp người không kỹ thuật.

② n8n — Low-code (node + chút logic)

Mỗi bước là một node; có node điều kiện (IF) để rẽ nhánh. Workflow lưu dạng JSON, có thể tự host:

# Pattern: CONDITIONAL (rẽ nhánh theo loại email)
[Webhook]  →  [OpenAI: phân loại]  →  [IF: loại == "Khiếu nại"?]
                                          ├─ TRUE  → [Slack: cảnh báo gấp #urgent]
                                          └─ FALSE → [Google Sheets: ghi log]
// Cấu hình rút gọn 1 node OpenAI trong n8n (JSON)
{
  "node": "OpenAI",
  "operation": "message",
  "prompt": "Phân loại email, chỉ trả về 1 từ: {{ $json.body }}"
}

③ LangChain — Code (Python, linh hoạt nhất)

Khi cần kiểm soát chặt / tích hợp sâu, viết bằng code. LangChain nối các bước thành "chain":

# Pattern: LINEAR — pip install langchain-anthropic
from langchain_anthropic import ChatAnthropic
from langchain_core.prompts import ChatPromptTemplate

model  = ChatAnthropic(model="claude-haiku-4-5-20251001")
prompt = ChatPromptTemplate.from_template(
    "Tóm tắt 2 câu và phân loại email sau: {email}"
)

chain = prompt | model            # nối: prompt → model thành 1 workflow
ket_qua = chain.invoke({"email": noi_dung_email})
# → bước tiếp: gọi Slack API gửi ket_qua.content
🧭 Chọn cấp độ nào?
No-code (Zapier)Việc đơn giản, cần nhanh, không muốn đụng code — nhưng tốn phí theo số lần chạy & khó tùy biến sâu.
Low-code (n8n/Make)Logic rẽ nhánh vừa phải, muốn tự host & tiết kiệm — đánh đổi bằng việc phải học giao diện node.
Code (LangChain)Cần kiểm soát chặt, tích hợp hệ thống riêng, test & version bằng git — đổi lại tốn công lập trình.
